# OGS Tournament Tracker

Very small static site that fetches public Online Go Server tournaments for a username.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## GitHub Pages

1. Push this repository to GitHub.
2. In the repo settings, enable GitHub Pages for the branch that contains `index.html`.
3. Use the published Pages URL and the site will work without a backend because the OGS API supports browser requests.

## Existing CLI

The original Python CLI is still available in `tournament_cli.py` if you want terminal output instead of the web UI.
